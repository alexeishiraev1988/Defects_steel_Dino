# Сегментация дефектов стального проката

Семантическая сегментация дефектов стали на основе **DINOv2 ViT-B/14 с DPT-декодером**.

Датасет: [Severstal Steel Defect Detection](https://www.kaggle.com/c/severstal-steel-defect-detection)  
Метрики на независимой выборке (5 332 изображения): **mIoU = 0.687**, **Mean Dice = 0.803**

---

## Быстрый старт

### 1. Клонировать репозиторий
```bash
git clone https://github.com/alexeishiraev1988/Defects_steel.git
cd Defects_steel
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Запустить приложение
```bash
python app.py
```

Веса модели (~341 МБ) скачаются автоматически с Яндекс Диска при первом запуске.
Также их можно скачать вручную:
-  model_exp17.pt (декодер, 10.9 МБ): https://disk.yandex.ru/d/RS5DNE-W83j7rA 
-  dinov2_exp17.pt (backbone, 330 МБ): https://disk.yandex.ru/d/TaKAGseYPGkExQ

После запуска приложения, откройте браузер: **http://127.0.0.1:7860**

По коду app.py интерфейс работает так:                                                                                                                                                                                           
  1. Загрузить изображение стальной поверхности (левая панель)                                                                                                                                                                     
  2. Нажать кнопку «Найти дефекты»                                                                                                                                                                                                 
  3. Справа появится изображение с цветными масками дефектов и текстовая статистика (% площади каждого класса)

## Архитектура модели

```
Вход 224×1400
      │
DINOv2 ViT-B/14  ── блоки 0–7: заморожены
      │           ── блоки 8–11 + norm: дообучены (28.4М параметров)
      │  промежуточные слои [3, 5, 8, 11]
      ▼
SegHeadDPT (декодер)
  proj(768→256) × 4
  fuse(1024→256)
  up1 ×2  (256→128)
  up2 ×2  (128→64)
  head(64→5)
      │
Маска сегментации 64×400
```

| Параметр | Значение |
|---|---|
| Входное изображение | 224 × 1400 пикселей |
| Классы | Фон + 4 типа дефектов |
| Параметров backbone | 86.6 М |
| Параметров декодера | 2.8 М |
| Скорость инференса | 13 FPS (RTX 5060 Laptop) |
